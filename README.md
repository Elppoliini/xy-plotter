# XY-plotter
LPC1549 receives Gcode from Mdraw and parses it to a form that a XY-plotter simulator can simulate the plotting.
M10  - value check
M11  - limit check
M2  - save pen position values
M5  - save dir,area,speed
speed
M1  - Set pen position
M4  - Set laser power
G28  - go to origin 
G1 e - go to position
